# Acesso remoto por SSH
## O que é SSH?
SSH (Secure Shell) é um protocolo utilizado para aceder remotamente a 
outro computador de forma segura através de uma rede.
## Que informação seria necessária para aceder a um servidor por SSH?
- Utilizador:
- Endereço IP:
- Porta:
- Palavra-passe ou chave:
## Exemplo de comando de ligação
ssh utilizador@endereco_ip
